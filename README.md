# BULWAR — Projekt bulwaru nad Wisłą (Cypel Czerniakowski)

Baza wiedzy projektu studenckiego dla przedmiotu **„Projektowanie bulwarów i promenad"** — SGGW.
Koordynator: **dr inż. Kinga Kimic**.

> Cel repozytorium: trzymać w jednym miejscu **całą wiedzę** projektu — dostarczone materiały,
> wyciągnięte z nich wnioski, decyzje projektowe, dobór roślinny i postęp prac nad planszami.

## Lokalizacja
Fragment **Cypla Czerniakowskiego** w Warszawie (nadrzeczna przestrzeń publiczna nad Wisłą).
Uwaga formalna: użytek ekologiczny; Dolina Środkowej Wisły to obszar Natura 2000 (PLB140004) →
priorytet gatunków rodzimych, unikanie inwazyjnych.

## Struktura repozytorium
```
00_brief/           – wymagania przedmiotu, kryteria oceny, wymogi formalne SGGW
01_analizy/         – analizy uwarunkowań (widokowe, komunikacja, dendrologia, fizjografia)
02_koncepcja/       – idea przewodnia, kompozycja plansz, program funkcjonalny
03_detal/           – detal indywidualny (meandrujące schody + kaskada + kieszeń roślinna)
04_rosliny/         – dobór roślinny wg stref hydrologicznych (kluczowa merytoryka)
05_plansze/         – layout i robocze wersje plansz B1
materialy-zrodlowe/ – surowe materiały dostarczane przez autorkę (mapy, zdjęcia, PDF-y)
notatki/            – dziennik decyzji projektowych i otwarte pytania
```

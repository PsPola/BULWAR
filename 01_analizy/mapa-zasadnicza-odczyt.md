# Odczyt mapy zasadniczej (wstępny)

Materiał źródłowy: mapa zasadnicza fragmentu Cypla Czerniakowskiego → `materialy-zrodlowe/mapa-zasadnicza.jpg`
Status: **wstępna interpretacja graficzna** (do potwierdzenia przez autorkę). Data: 2026-07-09.

## Co widać na mapie
- **Kształt terenu:** wydłużony, wąski **klin** biegnący po skosie, orientacja ok. NW–SE.
  Granica opracowania = linia kropka-kreska. → narracja liniowa („promenada wzdłuż"),
  linijka słońca kluczowa przez ten skos.
- **Głowica/cypel na północy (góra-lewo):** wyraźna **owalna pętla** z obwódką — istniejący
  element (plac / pętla / taras widokowy?). Naturalna **dominanta i punkt docelowy** trasy.
  W pobliżu etykieta „50607".
- **Pas ukośny z prostokątnymi, kreskowanymi blokami** wzdłuż środka — prawdopodobnie istniejące
  **utwardzenia / miejsca postojowe / schody**. Decyzja: zostawić / przekształcić / usunąć.
- **Zabudowa (środek-lewo):** kilka istniejących **budynków** (prostokąty z detalem) —
  uwzględnić w gospodarce terenem.
- **Rzędne wysokościowe** (liczby `+...`) rozsiane po całości → materiał do analizy zalewowej
  i wyznaczenia stref hydrologicznych A/B/C.
- **Prawa strona:** w dużej mierze **otwarta, pusta przestrzeń** (krzyże = siatka geodezyjna) →
  potencjał na strefę łąki / wypoczynku.
- **Lewa krawędź (przy wodzie):** szpaler drzew / zieleń istniejąca + linie brzegu.

## Wnioski wstępne dla projektu
- Wąski, długi teren → mocna oś kompozycyjna wzdłuż; poprzeczne „kieszenie" na zieleń i wypoczynek.
- Owalna pętla na północy = gotowy **punkt kulminacyjny** (kandydat na strefę widokową / spotkań).
- Skos NW–SE → nasłonecznienie zmienne wzdłuż trasy; zaplanować linijkę słońca dla stref siedzisk.
- Rzędne = podstawa doboru roślin wg stref zalewowych (patrz `04_rosliny/dobor-roslinny.md`).
